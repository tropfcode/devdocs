============
Print Stdout
============

print png to standard out

usage: python print_stdout.py > somefile.png




.. code-block:: python


    import sys
    import matplotlib
    matplotlib.use('Agg')
    import matplotlib.pyplot as plt

    plt.plot([1, 2, 3])

    if sys.version_info[0] >= 3:
        plt.savefig(sys.stdout.buffer)
    else:
        plt.savefig(sys.stdout)
