========
Log Axis
========

This is an example of assigning a log-scale for the x-axis using
`semilogx`.




.. image:: /gallery/scales/images/sphx_glr_log_test_001.png
    :align: center





.. code-block:: python


    import matplotlib.pyplot as plt
    import numpy as np

    fig, ax = plt.subplots()

    dt = 0.01
    t = np.arange(dt, 20.0, dt)

    ax.semilogx(t, np.exp(-t / 5.0))
    ax.grid()

    plt.show()
